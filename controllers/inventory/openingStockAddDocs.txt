FUNCTION NAME:

bulkOpeningStock

DESCRIPTION:

This API endpoint processes bulk opening stock uploads via CSV files for distributors. It downloads and parses CSV files, validates data integrity, checks for duplicates and existing inventory, fetches pricing information, creates inventory records across multiple godowns, generates stock transactions, and calculates RBP (Reward Bonus Points) for opening stock. The function includes advanced features like concurrent upload prevention, memory optimization, rate limiting, and comprehensive error handling.

The function ensures data consistency between Inventory, Product, Transaction, DistributorTransaction, and Distributor models while maintaining system performance and reliability.

STEP 13: Helper Functions


1. CSV Line Parser:
const parseCSVLine = (line) => {
  const result = [];
  let current = "";
  let inQuotes = false;
  for (let i = 0; i < line.length; i++) {
    const char = line[i];
    if (char === '"') {
      inQuotes = !inQuotes;
    } else if (char === "," && !inQuotes) {
      result.push(current.trim());
      current = "";
    } else {
      current += char;
    }
  }
  result.push(current.trim());
  return result;
};

2. Array Chunking:
const chunkArray = (array, size) => {
  const chunks = [];
  for (let i = 0; i < array.length; i += size) {
    chunks.push(array.slice(i, i + size));
  }
  return chunks;
};

3. Delay Function:
